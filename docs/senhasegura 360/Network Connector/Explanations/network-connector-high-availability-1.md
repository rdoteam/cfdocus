# About Network Connector High Availability (HA) 

The high availability of senhasegura Network ConnectorÂ is a feature of the product that works in Active-Passive mode. All data traffic is directed to the primary senhasegura and in case of failure, the secondary senhasegura comes into action and all data traffic is redirected to it.WarningIn case of switching between the Primary and Secondary Agent of the Network Connector, the user must restart the session to continue the operation.
